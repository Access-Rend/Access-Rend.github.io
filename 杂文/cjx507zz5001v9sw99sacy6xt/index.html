几乎是有记忆来第二次绝望
春夏秋冬

——纪念我直也是最喜欢的最后一个人

——慢下来读，也许我找的回当时感受

回到宿舍，才发现早上原来还剩了半盒奶没有喝——我想起来了，那是急着出门马上就被忘掉的半盒奶。不知道哪来的对一盒奶的愧疚，我把它心怀不安的喝掉了。

点起戒掉的烟，我又开始思念故地的玉兰花，几乎快忘却了的香味，和洁白的身子。

隔了两年快三年了，谁会知道我又开始写写自己的感情呢？可能这就是我自身的代偿吧，以弥补缺失的分享。不同的是，我也没抱希望任何人看它，也不会有华丽的辞藻了。相信我，看的都是少数人，大多也是不所熟识的人，也不会有太多人耐心读完，慢慢读完。

要怎么评价以前的感情呢？初恋应该是纯粹沉醉在甜言蜜语的状态中，单纯的享受这从小到大没有体会过的状态罢了。我犯了傻，也犯了病。但那个时候开始喜欢上的玉兰，确实是真的。白，香，甜，苦——把苦味留给愿意嚼的人，苦进心底，却发现着别人未曾拥有的乐趣和喜悦——我是唯一了解她的人。

可能，我喜欢的，是那种纯白的，像交响曲一样混合的香味吧？就像喜欢一个纯洁善良的，没有杂质的灵魂一样。

那么，我开始回忆了。

青灰色的春天

在很长一段的抑郁和焦虑中解放出来，我承认我走了极端——任何事情都考虑自己，不惜一切的为自己辩解自己没有错，对任何事都是。尽管如此，多时我还是戒不掉油然而生的愧疚和抱歉…也许天性如此吧？从小就被要求认同经书上各种老的观念，尽管理性到现在排斥着糟粕，难免忍不住强烈的负罪…我觉得只有极端可以把自己从深渊里拉出来了——那是高三，我必须把自己，靠自己，拉出来。我要活下去。

在平静了一些日子后，她慢慢活跃在我的视野。

知道什么最打动我吗？就是最单纯的善良了。渐渐我真的喜欢上她，那种渗透言行的，描绘不出的感觉。非要找个形容词，我觉得她是白的——现在也这么认为。(也许我在重复同样的过错)

她很温柔吧，几年下来我从来没有见过她发过脾气。之间也只是作为朋友交流着。能有一个共同爱好的朋友很不容易，她是为数不多被我当作同好的人，也是异性中唯一一个。

很早她让我摸摸她的头，半天憋出的话让我在心里窃喜，也在笑着她的模样。

也有一次在楼下聊天被当成男女朋友，让我有些惊讶。

但仅仅是作为朋友罢了。

高三了，我仍把自己封闭在自己的世界。突然来了一封她的回信——挺惊讶的，我没想到她会回我。但那个时候我开始不安了——我是否有能力去喜欢一个女生？我想，我很喜欢她，我是真正意识到我喜欢一个人不是因为激情或者别的很低层次的原因了。

心脏再一次开始悸动了。我窃喜，而没有告诉任何人，在那个青灰色的春天。

是的，我喜欢她呢。

盛夏

不经意熬过了高考，幸运的卡上了线，现在我进了一个被外人说作不错的学校，985嘛。

又到了少年们最纠结的志愿时刻。

一纸志愿断愁肠，两相对望一山哀。

她劝过我，也只是轻微的劝劝让我和她报一所学校…但出于专业考虑，我还是报了现在这所。很高兴，第一次有人在大事上在意我，是想和我一起念书嘛？原来我也不是会被人忽略掉的存在啊。虽然分数确实让我难受，但收到她的消息确实令我很开心，能在意我就很满足啦。

不在一个学校也无所谓嘛，s省和c省也隔着不远，真的想她了，大不了半个月就去看看嘛。我大概是个不怎么接触外界诱惑的人，所以对自己挺放心的。毕竟这个世界太乱。

不过还有一个长长的暑假呢，怎么安排呢？开始幻想着和她一起度过的时光了，止不住的间歇性偷笑，跟小朋友似的，说实话我知道很傻，但还是头一次这么做。

在不断的央求下，暑假我们有机会共同去做一个工作，每天在同一个地点。我挨着她，觉得那是最安心最快乐的时光了。尽管都在做自己的事，但还是说不出的happiness。

期待已久的那天到了，我可以和喜欢的人睡一起。可以享受一整夜抱着的权利，背着星空，说些故事。但当我讲完自己的故事后，有一种既放下了包袱也更加不安的感觉…很难受，空气凝固在喉管里，堵着那句“和我在一起好嘛”。无疑，那是最佳时机。我惶恐着，我觉得自己不配，也没有能力去用心喜欢一个搂着我说喜欢我的人——我有过抑郁，有过消极，现在想以一个阳光的姿态把青春的时光献给喜欢的人，却做不到了。

然后我点了烟…我丢不下心里包袱，过于沉重。两个人交心的话总会涉及到这些，我很自责将来将这些扔给他的人，那可能会是我。

我能给她一个快乐的时光吗？

越想，我脑子越要爆掉了…

然后我出去点了一根烟，悄悄的，流着无能泪。

但我至今记得打动我的，只是一个拥抱：“我喜欢你，喜欢就好了呀，为什么要纠结别的事情呢？”

是啊，我可以将过去的，责任不在我的包袱给丢掉，毫不负责的扔掉，去放心大胆的喜欢她啊。可是我能吗？

我想试试。

很开心，你让我有了唤醒已死心灵的勇气，不管再次堕落，还是成功，我很感激。谢谢你。

我决定想追一个女生了。而且她也喜欢我。

恰好也是我人生很重要的转变，值得铭记。

——如果有机会，我想好好爱你一次，一次就是很久。也许现在我做不到爱的程度，我想，我会的。

介于冷暖的秋天

爱又该如何体现呢？我的回答是像家人一样，那是自己最重要的人。宽容，温暖，有时也会有冲突，但是一直都是可以让心灵得到安慰和变得强大的地方。

潜意识里，她已经和家人一样，比家人更重要了。

我了解过柏拉图式恋爱，但只是不认同肉体肮脏罢了，比起肉体，我更渴望的是温暖。天生体质偏寒的缘故吧？即使抑郁着的时光我仍不停的渴求寻找着温暖。不经常有人和我说“辛苦了”，“还好吗？”，“没事啦”都会让我有想哭的冲动——也许是很少何人打交道？可即便这样，我内心依然不脆弱。

自古多情是男子，为什么会有这种奇怪的设定？？我都不解，也改不掉。

天天念念，层层点点，丝丝细雨，连绵不绝，千里江河，不如一见。

每天都期待相见，也没想过怎么玩，只是平淡的交流，拥抱，或者其他日常，我在幻想着。

架不住思念的折磨，一直和她商量着见面。我好想有一次不算约会的约会，来和她告白——用我等了十年有余的念想，换一个放手去爱的机会。

有一个故事：男人只是个教书的，在校门口画画却对女人一见钟情。本是荒唐的故事，本该只是持续一瞬间的心动，却有了下文——男人将亲手雕刻的核桃壳赠给了女人，核桃壳光滑细腻，上面刻画着一个眉清目秀的女子，像是戏剧中的脸，又像沉睡的女神。本该结束的故事却再度有了下文：他们在一起了。一个在校老师，和一个卫校的学生，就这么成了家人，过上了平平淡淡的生活。而见证者就是我们一家，他们的子嗣——也就是我。

我也存有美好的幻想，在小学便学起父亲悄悄的打磨着自己精挑细选的核桃壳。我想，我长大了一定要将自己最值得纪念的人刻在上面，像父亲一样。日复一日，那个核桃壳被断断续续的打磨得很光滑了。而我也长大到如今。

我觉得，这是我最值得喜欢，最值得去爱，去纪念的一个人。

就像奢侈着纯白的崇高理想一样。

我要把这个送给她，和她告白。


…


期间我们话很多，聊东聊西。她的每一次吐槽我都有好好听过，我也在用不很高明的方法，用自己的方式关心着她。可是为什么，我感觉不到呢？


…


她很久没有理我了。


…


七夕，我觉得很浪漫的日子，而且是我的生日，哪怕不在一弯月亮下，我也将把这个小玩意送给她，邮寄过去吧？我写了很多话，又撕掉很多…最后一封留了下来。

我调侃着生日在手术台上过七夕。——许久没有人给我过生日了，快十年了吧？我很期待着。

我真的很期待，吊瓶中流淌的仿佛不是术前液，是幸福的倒计时。会有人和我过生日吗？？


…


我想和她告白。

我想…听到一句七夕有关的，她在乎我的话。


…


我只想听她在我手术上台前的一句生日快乐。

我只，敢想她能在手术前关心我一句…客套话…

…

…

然而等来的是“哦”。

…

醒来已经是在病床上了，半夜了只剩下妈妈在旁边看着心率检测仪。似乎我血压不容易稳定。那机器一晚上叫着，每一声都刺痛着我。

她是不想理我了么？

她厌烦么？

是我自作多情吗？不会的…不然那晚算什么。

可是…除了看护我的妈妈，没有人问候过我啊…

我…我想起了以前，想起了一直以来生死冷暖无人问津。

我难受…但是身上插满了管子和线，我哭不了。而且我几年都哭不出来了。

我感觉又液体划过鼻腔，不知道是氧气带着水分滑落下来，还是泪腺仅存的反应。说不出为什么，就是很想哭罢了。

…

后来，我厚着脸皮去和好了。她说她挺懵逼的…

好吧，虽然喜欢她的蠢萌，不过楞到这地步也是没谁了…我总在怀疑自己的想法和感受…可能我就是很多心么？

核桃壳躺在我手心里，我仍期待着一个月前的约定。

秋冬过渡的不知名的季节，但没有秋老虎

两个城市，抵挡不了我坐动车去同一个城市的冲动，却被脚上的石膏阻挡了，牢牢地困我在轮椅
上。我想去，却去不了。

看着浮躁的大一新生，探索着校园。我仍在期待以后的学业和她。
没有什么比我的描述更蠢了。

我盼了三个月，终于要到了吗？

她说，她会和一个学妹去看演唱会，在国庆的倒数第二天，票都买好了，所以来不了了。

怎么会呢？晚一点走，或者早一点来，就可以了呀？为什么她会理直气壮的觉得是我的问题呢？还是说，这挑明了就是不想来么…

没有理由比她找的更好笑了。

…

期而不至，则是无信。

仁义智信礼，我最重视信与仁。

我确实做到了，遵守没一则诺言，我很骄傲这点。

…

太失望了，难受，生气…面对一个说喜欢自己却又变着法来推辞的人，我该作何思考？这个责任我不担。从三个月前，两个月前，一个月前我就一次又一次地询问着买票，去哪等…临时来了句不去了…试问谁能接受呢？

何况还是自己那么在意喜欢的人呢…除了痛心…

一次都没有，她没有安慰过，道歉也没有，甚至连不好意思都没有…哪怕只是客套话，我甚至怀疑自己接受的是假的耻感文化。

…

终于她好像退让了，说要来。我心里有些不是滋味…

算了吧，反正我消气也快，天生的好欺负。何况是她呢，有时候钝一点也没什么…也算萌的一种吧？

但是还是很难堪。

…

过去的都过去了，现在她要来才是最重要的。我拖着伤脚，努力的把清洁做了，整理好了自己，就期盼着那一天的到来。

虽然可能是我自己太作了吧，中间有些不愉快，但是还是很开心很期待呀。我想，怀揣着十一年的梦终于要送出去了，这一天一定是很重要的——尽管只是几个小时（她要赶火车走），也很让人幸福了。毕竟她愿意来看我嘛。

早上一早就没敢睡着，从六点等到十一点。她终于来了消息，说自己上地铁了，到了学校叫我。我问她在哪，也许是开玩笑，她说不告诉我。

然后她到了学校，说在东门了。

我急急忙忙关上写的代码，扔下半盒奶，挑起拐杖就往外赶，我怕等久了她会着凉，会饿着也说不定。总之，守时是很重要的，哪怕现在走不快。身上还揣着她要的草稿本——她说她想念高中好用的草稿本，让我帮她带三本。

路上，我接了电话，压抑住喜悦，逗着她。

“你在哪啊？”
“不告诉你。”我学着她的语气。
“那我找同学了接我了？”
“不告诉你。”

真的用赶来形容一点也不过分——大中午的谁看见一个杵着拐杖近乎一跳一跳的那个疯子兴许就是我了。

之后赶到了桥口，她回消息说，有同学来接她了。

说实话，我不知道怎么回答…

要怎么问呢…就是我不需要来了吗？

嗯

接近干脆的回答

…

那一刻，我不知道自己装着亲手刻了四个月的东西的盒子里，装了什么。

只是觉得好重…好重…

我真的没用，我想哭了…更没用的是，我哭不出来。

我想，用一个新的姿态去喜欢一个人，然后爱上她，戒烟是理所当然的吧？？

然后我又用烟，去代替我无声的哭泣。它使我能暂时忘掉孤独，也是我暂时的能倾诉的人——之所以戒不掉烟，可能是因为我需要它陪我说话吧…

…

之后负责推我轮椅的小哥陪着我坐了很久，错过了两顿饭，却任然复发的恶心。我知道我很接近抑郁的状态了，我记得这种感觉。很害怕，也很绝望，更多的是揪心的痛。

又来了，心痛到肉痛…怎么还是没好？我以为都好了呢…

所幸持续时间很短，我强忍着没让它发作，再一次，我用孤独的身体克服了常人克服不了的折磨，用时很短，比上一次短多了。我比克服不了的人意志力强太多了，真佩服自己。

…

而每听到一次熟悉的脚步，我就会抬起头来看一眼：综合性学校的校园这么大，我也渴望见到一次，只是想看见一次。

然而我回她的消息是：我真的不想再看到你了。

我很确信，最后一次脚步声是她经过的，因为我透过模糊的眼睛和头发看到了熟悉的背影，熟悉一个人靠背影动作来分辨，只要有心，并不会很难。

…

纯白的冬

我在一个认识不久交情很浅的同学身上靠了两个小时，那时我没力气，体质偏寒，行人们都穿着卫衣了，我穿的却很少——急着出门，忘了看天气。

体会着寒冬才有的风的凌冽，做出发抖的动作。

躺在路边的草坪，被当成尸体问了好多次了“同学你没事吗？”慌忙道：“啊谢谢，不好意思我在睡觉…”谁都看得透的谎言。

最丢人的姿态。然而我不在乎了。我想要的只是这一片刻的宁静和放松，以及放松不了的，很破碎的心。躺在草坪上，已经很优雅了吧？

我很喜欢冬天，不知道为什么，下雪很令我兴奋，虽然没见过两次雪的模样。但我依然期待着，渴望着，雪花纯白的颜色。

就还像渴望着什么若有若无的东西一样。

靠在小哥身上干嚎了一会，我是真的尽力了。试了几年没哭出来，真的很失败，我觉得会更憋得令人扭曲。小哥说我很迷，这种扭曲的人他是头一次见。

…

我又想起了玉兰，她就是在这种春秋冷雨之际咲放的。那种不可企及的香啊，纯白的身姿。令人流泪的味道。

我想起了一个人，那个每次都陪着我我，语言上安慰不了却在关键时刻实实在在陪着我的妹妹。我想告诉他，我难受了，我真的很没用。也许她会摸一摸我的头？

可惜她不在此地，而且她也要远走了。也许久没有找过她了，我想我要适应。

倾诉就打住，就连说话的对象也没有。我想点开华为助手，可是我嫌他傻。

…

如果她看得到的话，我还是有些想说的话吧。

亲口告诉我，你是否在乎我。

我很用心，我想好好待你…

我似乎还在期待着道歉？如果真正有人喜欢我，我怎么可能在意这一切呢？

下决心放手了，只是因为你不回答，我几乎快确信，我是一直被玩弄的人。

哭的，也仅仅是和过去串联在一起的自己。无关他人，请不要说我软弱。

最后，我仍会喜欢这个名字很久。

…

似乎也是自说自话吧…没办法我可能就是悲哀至此。个人主义的悲剧吧？

…
…

没办法，我得回去做我的高数了。

寝室里怎么躺着半盒牛奶？
原来是走得太急剩下的。

啊，该吃药了，可是没吃饭怎么吃药呢…

管他的，这个状态估计又是几天都吃不了饭了，买点葡萄糖喝吧。

下周回去看玉兰花吧？要开了估计。
